# 02. Product Requirements Document


Project: MapleWealth, Personal Finance Dashboard
Owner: Master
Version: 1.0
Primary reference: Personal Financial Master Plan (Canada)
Core rule: Emergency Fund -> TFSA -> FHSA -> RRSP. Do not recommend trading, market timing, options, forex, meme stocks, or speculation.


## Users
Primary user: Master, age 30, Canada, software developer, salary currently $37,000 CAD, monthly take-home about $2,800, monthly expenses about $1,500, monthly savings capacity about $1,300.

## Financial Baseline
- Emergency fund: CIBC Savings, current $1,000, target $5,000 minimum and $8,000 ideal.
- Vacation fund: Neo Savings, current $3,000, must not be invested.
- TFSA: Wealthsimple Self-Directed, initial $10, recurring $50/month, primary ETF XEQT.
- Wealthsimple chequing accounts: 2 accounts, created partly due to promotion entries. Treat as cash accounts, not investment strategy.

## MVP Features
1. Dashboard summary
2. Account management
3. Net worth tracker
4. Emergency fund tracker
5. Vacation fund tracker
6. TFSA tracker
7. ETF holdings tracker
8. Dividends and DRIP tracker
9. Recurring deposits tracker
10. Projection calculator
11. Monthly review report
12. CSV import
13. Manual transaction entry
14. Rules engine based on personal financial plan

## Acceptance Criteria
- User can enter all accounts and balances.
- User can see current net worth and progress to $100,000.
- User can see emergency fund progress to $5,000 and $8,000.
- User can track TFSA contribution history and avoid overcontribution.
- User can track $50/month recurring TFSA deposit.
- User can record XEQT purchases, dividends, and DRIP reinvestments.
- System warns when investing is prioritized before emergency fund basics are handled.
- System can generate monthly summary with actions.
