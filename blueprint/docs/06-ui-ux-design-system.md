# 06. UI/UX Design System


Project: FinanceOS Canada, Personal Finance Dashboard
Owner: Master
Version: 1.0
Primary reference: Personal Financial Master Plan (Canada)
Core rule: Emergency Fund -> TFSA -> FHSA -> RRSP. Do not recommend trading, market timing, options, forex, meme stocks, or speculation.


## UI Direction
Professional SaaS dashboard. Clean, dark-mode friendly, dense enough for power users, simple enough for monthly review.

- Dividend Reinvestment Card
- Savings Rate Card
- Salary Milestone Card
- Action Required Card

## UX Rules
- Show next best action, not just charts.
- Avoid red/green performance addiction.
- Default investment view should be long-term, not daily.
- Make emergency fund status visually obvious.
- Show warnings before speculation or misaligned asset choices.

## Navigation
Left sidebar on desktop. Bottom nav or compact drawer on mobile.
